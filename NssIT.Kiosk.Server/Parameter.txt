xxxxx
* Use "xxxxx" (5"x") to indicate end of Parameters setting.
* Use "=" to assign value
* A Space " " is not allowed in parameter setting.

* PayMethod : C for Cash, D for Paywave(Credit Card); C,D mean Cash and Paywave ;
    D for Paywave/Credit Card/Debit Card
	C for Cash
	T for Payment Gateway / e-Wallet (like Boost & TnG)
	T,C,D for e-Wallet, Cash, Paywave

	Use "," as separator. Like C,D

* WebServiceURL :  Web Service URL for any data transaction.
* IsDebugMode : This is an optionanl parameter. Remove this parameter if not necessary.
			: Example -> IsDebugMode=true

* LocalServerPort : Port number used for data receiving at Local Server (NssIT.Kiosk.Server.exe / Windows Service)

* AppGroupCode	: Optional Parameter.
				: Default value is Melaka.
				: If this parameter has not set, this value is default to Melaka (Melaka Sentral)
				: The Valid value is Melaka / Larkin / Gombak / Klang.

* IsBoardingPassEnabled	: Optional Parameter.
						: Default value is false. 
						: Allow system to print Boarding Pass if true.

* BTnGMinimumWaitingPeriod	: 'Boost/Touch n Go' minimum waiting period in seconds. 
							: This period used for response waiting after 2D Barcode has shown.
							: Default and Minimum value is 90.
